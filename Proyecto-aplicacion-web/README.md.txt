# Proyecto Aplicación Web

Sistema web para la gestión de clientes, inventarios y ventas.

## Tecnologías utilizadas

- ReactJS
- Node.js
- Express
- Axios
- Swagger

## Funcionalidades

- Gestión de clientes
- Gestión de inventario
- Registro de ventas
- Reportes
